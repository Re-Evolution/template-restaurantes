Politica de Privacidade

Ultima atualizacao: Fevereiro de 2026

O restaurante O Sem Nome compromete-se a proteger a privacidade dos seus clientes e visitantes do nosso website. Esta Politica de Privacidade explica como recolhemos, utilizamos e protegemos os seus dados pessoais.

1. Responsavel pelo Tratamento de Dados

O responsavel pelo tratamento dos dados pessoais recolhidos atraves deste website e:

2. Dados Pessoais que Recolhemos

Podemos recolher os seguintes tipos de dados pessoais:

2.1 Dados fornecidos diretamente por si:

  • Nome completo
  • Endereco de email
  • Numero de telefone
  • Data e hora da reserva pretendida
  • Numero de pessoas
  • Preferencias alimentares ou alergias (quando indicadas)
  • Mensagens ou observacoes adicionais

2.2 Dados recolhidos automaticamente:

  • Endereco IP
  • Tipo de navegador e dispositivo
  • Paginas visitadas e tempo de permanencia
  • Data e hora de acesso

3. Finalidade do Tratamento

Os seus dados pessoais sao tratados para as seguintes finalidades:

  • Gestao de reservas: Processar e confirmar as suas reservas no restaurante
  • Comunicacao: Entrar em contacto consigo relativamente a reservas ou pedidos de informacao
  • Melhoria do servico: Analisar a utilizacao do website para melhorar a experiencia do utilizador
  • Obrigacoes legais: Cumprir com obrigacoes legais e regulamentares aplicaveis

4. Base Legal para o Tratamento

O tratamento dos seus dados pessoais baseia-se em:

  • Execucao de contrato: Quando necessario para processar a sua reserva
  • Consentimento: Quando nos autoriza expressamente a comunicar consigo
  • Interesses legitimos: Para melhorar os nossos servicos e operacoes
  • Obrigacao legal: Quando exigido por lei

5. Partilha de Dados

Os seus dados pessoais nao sao vendidos a terceiros. Podemos partilhar os seus dados com:

  • Prestadores de servicos que nos auxiliam nas operacoes (ex: sistemas de reservas)
  • Autoridades publicas, quando exigido por lei

6. Retencao de Dados

Os seus dados pessoais sao conservados apenas durante o tempo necessario para as finalidades para as quais foram recolhidos:

  • Dados de reservas: 2 anos apos a data da reserva
  • Dados de navegacao: 1 ano
  • Dados para cumprimento de obrigacoes legais: conforme exigido por lei

7. Os Seus Direitos

Ao abrigo do Regulamento Geral sobre a Protecao de Dados (RGPD), tem os seguintes direitos:

  • Direito de acesso: Obter confirmacao sobre se tratamos os seus dados e aceder aos mesmos
  • Direito de retificacao: Corrigir dados inexatos ou incompletos
  • Direito ao apagamento: Solicitar a eliminacao dos seus dados em determinadas circunstancias
  • Direito a limitacao: Limitar o tratamento dos seus dados
  • Direito a portabilidade: Receber os seus dados num formato estruturado
  • Direito de oposicao: Opor-se ao tratamento dos seus dados

Para exercer qualquer destes direitos, contacte-nos atraves do email: geral@osemnome.pt

8. Seguranca dos Dados

Implementamos medidas tecnicas e organizativas adequadas para proteger os seus dados pessoais contra acessos nao autorizados, perda ou destruicao, incluindo:

  • Encriptacao de dados em transito (HTTPS)
  • Acesso restrito aos dados pessoais
  • Monitorizacao regular de seguranca

9. Cookies

Este website pode utilizar cookies para melhorar a sua experiencia de navegacao. Os cookies sao pequenos ficheiros de texto armazenados no seu dispositivo. Pode gerir as preferencias de cookies atraves das definicoes do seu navegador.

10. Alteracoes a esta Politica

Podemos atualizar esta Politica de Privacidade periodicamente. Quaisquer alteracoes serao publicadas nesta pagina com a data de atualizacao revista.
